PYTHON FULL STACK

1. What is Python?

Python is a high-level programming language.
It is easy to learn and write.
It is widely used for web development and automation.

2. What is Full Stack Development?

Full Stack Development includes frontend and backend development.
Frontend handles the user interface.
Backend manages data and business logic.

3. What is HTML?

HTML stands for HyperText Markup Language.
It creates the structure of web pages.
Headings, tables, and forms are created using HTML.

4. What is CSS?

CSS stands for Cascading Style Sheets.
It is used to design web pages.
Colors, fonts, and layouts are controlled using CSS.

5. What is JavaScript?

JavaScript makes web pages interactive.
It can validate forms and handle events.
It runs directly in the browser.

6. What is Django?

Django is a Python web framework.
It helps build secure web applications.
It follows the MVT architecture pattern.

7. What is Flask?

Flask is a lightweight Python framework.
It is simple and flexible to use.
It is suitable for small and medium projects.

8. What is MySQL?

MySQL is a relational database management system.
It stores and manages application data.
It uses SQL for database operations.

9. What is Git?

Git is a version control system.
It tracks changes made to source code.
It helps developers collaborate efficiently.

10. What is GitHub?

GitHub is a cloud-based platform for Git repositories.
It allows developers to share and manage code.
It supports teamwork and project collaboration.
